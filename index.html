Download & Install

Choose your browser and follow the installation instructions:

⬇️ Download JanitorTension (.CRX)

Installing on Chrome (Annoying But Works)

  1. Download the .crx file above.
  2. Rename the file from .crx to .zip.
  3. Extract the contents of the zip to a folder.
  4. Go to chrome://extensions and enable Developer Mode (top right).
  5. Click “Load unpacked” and select the folder you extracted.
  6. Done. Ignore Chrome’s warnings — it’s just salty it’s not from the Web Store.

Installing on Opera GX (Much Easier)

  1. Download the .crx file above.
  2. Open the Opera GX extensions page: opera://extensions
  3. Enable Developer Mode in the top right.
  4. Just drag and drop the .crx into the page. Opera will prompt you to install.
  5. Click Install and you're set!
